# financial-crisis-early-warning-model
Anomaly detection model flagging 2008 and 2020 crises 

## Overview

Developed a hybrid machine learning framework to identify early warning signals of financial crises using macroeconomic and financial market indicators. The model combines deep learning, anomaly detection, and statistical validation techniques to detect periods of financial stress, including the 2008 Global Financial Crisis and the 2020 COVID-19 market disruption.

## Key Features

- Hybrid anomaly detection using Isolation Forest, LOF, and One-Class SVM
- LSTM model for time-series forecasting
- SHAP analysis for model interpretability
- Z-score statistical validation
- Ensemble approach to improve crisis detection

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- TensorFlow/Keras
- SHAP
- Google Colab

## Results

- Successfully identified major historical financial crises
- Hybrid model achieved:
  - Accuracy: **88%**
  - Precision: **92%**
  - Recall: **84%**
  - ROC-AUC: **0.88**
